# FYN-X Companion Droid

FYN-X is an in-universe Star Wars companion droid implemented using Ollama and a JSON-backed retrieval-augmented generation (RAG) system.

## Features
- Strong in-character system prompt (protocol droid–style)
- JSON-based archival knowledge store
- Deterministic tag-based retrieval
- Prompt injection for contextual grounding
- Designed for future embedding-based RAG

## Requirements
- Python 3.11
- Ollama
- Windows, Linux, or macOS

## Running
```cmd line
python -m fynx_run
